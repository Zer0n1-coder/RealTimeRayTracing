## 简介
关注实时渲染的开发人员应该都知道《Real-Time Rendering》这本书号称实时渲染界的百科全书，书中内容虽然没有详细展开任何一项实时渲染有关技术的内容，却给出了大量的深入了解相关技术的途径。书中内容几乎涵盖了书本出版前的所有的最新技术，如果想了解实时渲染技术的全貌，这本书足以满足你。即便你想深入了解相关技术，这本书作为实时渲染技术目录的存在，也能引导你去如何深入了解相关技术。
